Metodo Scamper
Actividad "SCAMPER"

Utiliza el método SCAMPER para validar tu idea. Aplicar el método no significa que necesariamente vayas a cambiar cosas en tu propuesta de valor o negocio, solo ver que existen posibilidades de hacer las cosas de forma diferente y comprobar que la forma en que haz diseñado tu producto o servicio es la mejor que existe. Para aplicar el método haz lo siguiente:

  1. Piensa, ¿Cómo SUSTITUIR cosas en tu producto?, agrega 3 propuestas, cualquier propuesta es válida.

  2. Piensa, ¿Cómo SUSTITUIR cosas en tu producto?, agrega 3 propuestas, cualquier propuesta es válida.

  3. Piensa, ¿Qué partes de tu producto se pueden adaptar para hacerlo mas valioso para algunos clientes?, agrega 3 propuestas.

  4. Piensa, ¿Qué partes de tu producto se pueden modificar (cambiar por algo diferente) para hacerlo más atractivo para tus clientes? agrega 3 propuestas.

  5. Piensa, ¿Se pueden dar otros usos a tu producto o servicio?, por ejemplo coca cola se puede utilizar para limpiar motores, agrega tres propuestas.

  6. Piensa, ¿Existen partes, características, secciones o beneficios de tu producto que se puedan eliminar? Esto podría hacerlo más barato, más ligero, más accesible o mejor para tus clientes. Agrega 3 propuestas.

  7. Piensa, ¿Se puede organizar de forma diferente el proceso de producción. distribución, etc. de tu producto o servicio para mejorarlo? agrega 3 propuestas.

Pon tus resultados en un documento word o pdf y súbelos en este espacio. Te incluimos una plantilla que podría ayudarte, además revisa el contenido "Scamper" para más detalles y ejemplos.
